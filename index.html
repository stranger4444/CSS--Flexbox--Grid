Демонстрація CSS, Flexbox та Grid

CSS

CSS використовується для стилізації елементів сторінки — кольори, шрифти, відступи та інше.

Flexbox

Flexbox допомагає розміщувати елементи в ряд або колонку та автоматично вирівнювати їх.

Grid

CSS Grid дозволяє створювати складні макети у вигляді сіток — дуже зручно для основного контенту сторінки.
